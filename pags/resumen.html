La manera de agregar el estilo en linea es escribiendo directamente en el mismo código html, por ejmplo: "h1 style="color: blueviolet; text-transform:uppercase;">Hola, Introducción a CSS3" y se le agregan reglas separando solo con el punto y coma (;)

Para escribirlo de manera interna pero no en linea es creando las etiquetas debajo.

Por ejemplo : abrimos la etiqueta de style y escribimos las reglas. style regla 1; regla 2; font-size; p{color: verde;} y cerramos la etiqueta de style de nuevo

De manera externa

Para añadir el estilo de manera externa se tiene que crear un archivo .css y linkearlo al html. Este es el metodo mas adecuado ademas de ser mas sencillo por la posibilidad de crear estilo para cada seccion evitando tener un solo codigo muy extenso.
